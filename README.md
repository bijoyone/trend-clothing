# trend-clothing
## Large &amp; Complex E-Commerce Clothing Store  
 
 - React
 - React Hooks
 - React Routing
 - Reducer
 - React Context
 - Redux
 - Reselect library for Redux
 - Redux Persist
 - Redux-Thunk
 - Redux-Saga
 - Firebase for Authentication and Authorization 
 - Firebase for Database Storage
 - SCSS
 - CSS in JS with styled-components
 - Deploying with Netlify
